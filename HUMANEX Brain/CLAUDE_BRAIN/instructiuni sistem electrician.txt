# CONTEXT SISTEM ELECTRICIAN ⚡ (FĂRĂ LOGIN)

## Scop

Construim un sistem simplu și profesional pentru o firmă de electricieni unde:

* șeful de echipă introduce zilnic raportul
* patronul primește automat email cu raportul
* totul este rapid și fără login

---

## Ce este deja implementat

### Backend

* Supabase configurat
* Tabel: `daily_reports`

### Coloane:

* project_name (text)
* materials_used (text)
* operations_done (text)
* notes (text)
* equipment_used (text, optional)
* equipment_hours (number, optional)
* report_date (date)

---

### Automatizare (n8n)

Workflow complet:

1. Schedule Trigger (ora 18:00)
2. HTTP Request → Supabase
3. Filtrare după ziua curentă
4. Code Node → HTML profesional
5. Email → trimis către patron

✔ Funcționează complet

---

## Ce trebuie construit (FOCUS)

### 1. Pagina de introducere raport (FOARTE IMPORTANT)

O pagină simplă (fără login) unde șeful de echipă poate completa:

* Nume proiect
* Materiale folosite
* Operații efectuate
* Note / observații
* Utilaje folosite (optional)
* Ore utilaje (optional)

---

## Cerințe UX (CRITIC)

* Mobile-first (folosit pe telefon)
* Foarte simplu
* Form mare, clar
* Buton mare: “Trimite raport”
* Feedback după submit (ex: “Raport trimis”)

---

## Comportament

La submit:

* trimite datele în Supabase (insert)
* setează automat:
